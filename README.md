Тренировка:

разработать консольную программу, которая открывает 2 изображения одинакового
размера (in1.jpg, in2.jpg) и делает с ними следующие операции:

1. вычисляет попиксельно сумму двух изображений и формирует выходное
   изображение out1.jpg
2. вычисляет попиксельно среднее арифметическое двух изображений и формирует
   выходное изображение out2.jpg
3. вычисляет попиксельный максимум двух изображений и формирует out3.jpg
